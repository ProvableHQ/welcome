# Develop on Aleo with Leo

## 1. Build Guide

### 1.1 Install Git:

**[bit.ly/start-git](https://bit.ly/start-git)**

### 1.2 Install Rust:

**[bit.ly/start-rust](https://bit.ly/start-rust)**

### 1.3 Install Leo

```bash
curl -L https://raw.githubusercontent.com/AleoHQ/workshop/master/install.sh | sh 
```
