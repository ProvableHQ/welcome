### `leo logout`
Deletes the locally stored credentials file. 
```
leo logout
```
```bash title="console output:"
     Logout success
```

### USAGE
```
leo logout [FLAGS]
```

### FLAGS
* `-V`, `--version` - Prints version information
* `-h`, `--help` - Prints help information
