```bash title=ENDPOINT
GET /testnet/peers/all
```

Returns the peers connected to the node.

### Arguments

None

### Response

| Parameter | Type  |          Description           |
|:---------:|:-----:|:------------------------------:|
| `result`  | array | The list of connected peer IPs |
