```bash title=ENDPOINT
GET /testnet/peers/all/metrics
```

Returns the peers and their types connected to the node.

### Arguments

None

### Response

| Parameter | Type  |                  Description                  |
|:---------:|:-----:|:---------------------------------------------:|
| `result`  | array | The list of connected peer IPs and node types |
